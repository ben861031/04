# 公文逾期稽催管理系統 (Document Tracking System)

這是一個純前端執行的「公文逾期稽催管理系統」，透過瀏覽器即可將公文系統產出的 Excel 清單進行匯入、過濾、重新排版與統計，幫助使用者快速辨識並追蹤逾期未歸檔的公文。

## ✨ 系統功能

1. **Excel 資料匯入與解析**
   - 支援拖曳或點選上傳 `.xlsx`, `.xls` 檔案。
   - 自動將資料讀取並暫存於本機（LocalStorage），重新整理網頁資料不會遺失。
   - 介面會自動顯示「最後載入檔案」的名稱與時間，方便辨識版本。

2. **智慧公文分類與手動校正**
   - **自動判斷**：系統會自動根據「收文日期」與「應歸檔/限辦日期」的處理天數差距，預設判斷該公文為「收文」或「發文」。
   - **自訂基準**：使用者可自訂判斷收發文的天數門檻（預設為 14 天），設定會自動記錄。
   - **手動調整**：若系統判斷有誤，可在表格中利用彩色標籤的下拉選單進行手動覆寫，手動修改的結果同樣會自動記憶在瀏覽器中。

3. **動態稽催基準日與逾期計算**
   - 使用者可自訂「稽催基準日」（預設為當天）。
   - 系統會依據基準日，與公文的「應歸檔日期」等欄位進行比對，自動計算並標示「已逾期天數」。
   - 逾期案件的整行會以「紅色底色與紅字」高亮顯示，以作警示。

4. **多維度資料篩選與搜尋**
   - **關鍵字搜尋**：可快速搜尋表單編號、主旨、承辦人等任意內容。
   - **部門分頁**：支援讀取 Excel 的工作表（Sheets），可選擇特定部門分頁或合併查看「全部部門」。
   - **收發文篩選**：可快速切換只看「收文」或「發文」。
   - **欄位排序**：點擊表格標題即可針對該欄位進行升降冪排序。

5. **統計與報表匯出**
   - **即時統計**：上方儀表板即時顯示目前篩選條件下的「收文逾期數」與「發文逾期數」。
   - **複製查詢結果**：一鍵將當前表格內容複製到剪貼簿，且附帶完整的格式與寬度排版，可直接貼上至 Email 寄出。
   - **匯出 Word**：自動產生包含公司抬頭（環興科技股份有限公司 稽催清冊）的 Word 報表（`.doc`）。
   - **匯出 Excel**：將整理與篩選過後的公文清單重新匯出為 `.xlsx` 檔案。

## 🚀 使用方式

由於本系統為純前端架構（Vanilla JS + HTML + CSS），不需要額外架設伺服器或是安裝資料庫，使用方式非常簡單：

1. **開啟系統**：
   直接使用瀏覽器（推薦使用 Chrome 或 Edge）雙擊開啟專案資料夾中的 `index.html`。
2. **匯入資料**：
   將公文系統匯出的 Excel 檔案拖曳至上方「資料匯入與篩選」區塊，或點擊「選擇檔案」上傳。
3. **設定條件**：
   - 確認「稽催基準日」是否正確。
   - 利用「公文類型」或「部門分頁」快速切換您想查看的視角。
   - 如果有找尋特定案件的需求，可使用「關鍵字搜尋」。
4. **檢查與修改類型**：
   查看清單中的「最終類型」欄位，若有不符的案件，可以直接透過下拉選單切換「收文/發文」。
5. **產生報表**：
   確認畫面上的資料無誤後，可利用右側的按鈕：
   - 點擊 **[複製查詢結果]** 貼至信件或通訊軟體。
   - 點擊 **[匯出 Word]** 作為書面報告。
   - 點擊 **[匯出 Excel]** 作為電子檔歸檔。

### 🧹 資料清除
如果想結束作業，或者要載入完全無關的全新檔案，可以點選 **[清除資料]**，這會將畫面上與暫存在瀏覽器中的資料完全清空。
