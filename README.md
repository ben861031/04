# 公文逾期稽催管理系統 (Document Tracking System)

這是一個純前端執行的「公文逾期稽催管理系統」，透過瀏覽器即可將公文系統產出的清單進行匯入、過濾、重新排版與統計，幫助使用者快速辨識並追蹤逾期未歸檔的公文。本系統完全在本地端運行，兼具企業級介面的專業度與資料絕對的安全性。

## 核心系統特色

### 1. 現代化專業行政介面 (Professional UI)
- **頂部導覽列架構**：採用現代化企業軟體的頂部導覽列設計，將系統動作與資料區塊分離，視覺清爽無負擔。
- **外部系統無縫串聯**：導覽列左側內建「檔管作業」與「原系統稽催清冊」的快速連結，方便承辦人隨時切換回官方系統進行比對或後續作業，且皆採新分頁開啟，不干擾當前工作進度。
- **極簡嚴謹風格**：捨棄多餘的圖示與花俏圓角，以乾淨俐落的網格、收斂的色彩計畫，呈現正式系統應有的莊重與專業感。
- **優雅的警示設計**：對於逾期案件，捨棄刺眼的滿版紅底，改採內斂的單元格紅字標示，降低視覺焦慮。

### 2. 雙軌資料匯入模式 (Data Import)
- **智慧匯入 (從剪貼簿貼上)**：無須下載 Excel！直接在舊有公文系統中反白複製表格，點擊「智慧匯入」後貼上，系統便能自動解析網頁表格並無縫轉換，大幅縮短作業時間。
- **全螢幕拖曳匯入 Excel**：將 Excel 檔案（`.xlsx`, `.xls`）拖曳至網頁任意角落，即可瞬間完成解析匯入。

### 3. 高效且安全的離線儲存 (Local Storage)
- **F5 重新整理防護**：無論是透過 Excel 還是「智慧匯入」的資料，都會被安全地轉換為穩定格式並暫存於本機瀏覽器（LocalStorage）中。即使不小心關閉分頁或按下 F5 重新整理，所有資料與進度皆會完美還原。
- **零後端架構**：所有資料皆在使用者本機端進行解析與運算，無需上傳至任何伺服器，100% 保障機密公文資訊的安全。

### 4. 智慧分類與彈性篩選
- **自動化公文判斷**：系統根據處理天數差距，自動預判「收文」或「發文」，並提供可自訂的判斷門檻天數。
- **強制手動校正**：若系統判斷有誤，可直接透過下拉選單手動切換最終類型，系統會自動記憶您的修改。
- **多維度檢視**：支援部門分頁切換、收發文快速過濾，以及支援關鍵字的全局搜尋功能。

### 5. 報表輸出與分享
- **即時數據儀表板**：頂部儀表板即時統計當下條件的收/發文逾期總數。
- **一鍵複製表格**：點擊「複製查詢結果」，直接至 Outlook 或 Line 按下 `Ctrl+V`，即可貼上格式完美的稽催表格。
- **匯出 Word / Excel**：提供標準化的 `.doc` 書面報表（含公司抬頭）與 `.xlsx` 電子歸檔檔案下載。

## 🚀 使用說明

由於本系統為純前端架構，**不需要架設伺服器**，只要有瀏覽器即可開始作業：

### 步驟一：開啟系統
使用瀏覽器（推薦 Google Chrome 或 Microsoft Edge）雙擊開啟 `index.html`。

### 步驟二：匯入資料
位於畫面右上角，您可以選擇兩種匯入方式：
1. **[匯入 Excel]**：點選按鈕或將檔案直接拖曳進網頁畫面中。
2. **[智慧匯入]**：若不想下載檔案，可直接複製原系統的表格，點擊此按鈕後在視窗中貼上，讓系統自動為您解析。
> 系統會於導覽列顯示「最後載入檔案」的狀態，並在您下次開啟時自動還原資料。

### 步驟三：條件設定與操作
- 確認「**稽催基準日**」（預設為今日），系統會依此自動計算「已逾期天數」。
- 利用工具列的「**公文類型**」、「**部門分頁**」與「**關鍵字搜尋**」來聚焦需要追蹤的案件。
- 檢查「**最終類型**」欄位，若判斷不符，可直接點擊下拉選單手動覆蓋。

### 步驟四：輸出與匯出
確認無誤後，使用右方的匯出按鈕：
- **[複製查詢結果]**：適合快速透過 Email 或通訊軟體回報。
- **[匯出 Word]**：適合列印成書面報告（自動附上標準抬頭）。
- **[匯出 Excel]**：適合留存數位檔案備查。

### 🧹 重置系統
若要處理下一批全新的資料，請點擊 **[清除資料]** 以清空畫面與暫存，避免新舊資料混淆。
