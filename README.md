# 公文逾期稽催管理系統 (Document Tracking System)

這是一個純前端執行的「公文逾期稽催管理系統」，透過瀏覽器即可將公文系統產出的 Excel 清單進行匯入、過濾、重新排版與統計，幫助使用者快速辨識並追蹤逾期未歸檔的公文。本系統完全在本地端運行，確保了資料的安全性與隱私。

## ✨ 系統核心功能

1. **Excel 資料匯入與解析 (支援拖曳上傳)**
   - 支援拖曳或點選上傳 `.xlsx`, `.xls` 檔案。
   - **本地暫存**：自動將載入的資料暫存於本機瀏覽器（LocalStorage），即使不小心重新整理網頁，資料與進度也不會遺失。
   - **載入紀錄**：介面會自動顯示「最後載入檔案」的名稱與時間，方便辨識當前使用的版本。

2. **智慧公文分類與手動校正**
   - **自動判斷**：系統會根據公文的處理天數差距，預設判斷該公文為「收文」或「發文」。
   - **自訂基準**：使用者可自訂判斷收發文的天數門檻（預設為 14 天），設定會自動記錄。
   - **手動調整**：若系統判斷有誤，可在表格中利用彩色標籤的下拉選單進行手動切換（收文為藍色，發文為黃色），手動修改的結果同樣會自動記憶。

3. **動態稽催基準日與逾期計算**
   - 使用者可自訂「稽催基準日」（預設為當天）。
   - 系統會依據基準日，與公文的「應歸檔/限辦日期」進行比對，自動計算並標示「已逾期天數」。
   - 逾期案件整行會以「紅色底色與紅字」高亮顯示，以作強烈警示。

4. **多維度資料篩選、排序與搜尋**
   - **關鍵字搜尋**：可快速搜尋表單編號、主旨、承辦人等任意內容。
   - **部門分頁支援**：支援讀取 Excel 的多個工作表（Sheets），可選擇特定部門分頁或合併查看「全部部門」。
   - **收發文篩選**：可快速切換只看「收文」、「發文」或「全部公文」。
   - **欄位動態排序**：點擊表格標題即可針對該欄位進行升降冪排序。

5. **統計與多樣化報表匯出**
   - **即時數據儀表板**：上方儀表板即時顯示目前篩選條件下的「收文逾期數」、「發文逾期數」、目前分頁及最後載入檔案資訊。
   - **複製查詢結果**：一鍵將當前表格內容複製到剪貼簿，並自動處理好字體（標楷體）、邊框與寬度排版，可直接完美貼上至 Email 寄出。
   - **匯出 Word 報表**：自動產生包含公司抬頭（環興科技股份有限公司 稽催清冊）的 Word 報表（`.doc`），適合書面列印與報告。
   - **匯出 Excel**：將整理與篩選過後的目前公文清單，重新匯出為乾淨的 `.xlsx` 檔案，方便電子檔歸檔與二次處理。

## 🚀 使用方式

由於本系統為純前端架構（Vanilla JS + HTML + CSS），**不需要額外架設伺服器或是安裝資料庫**，只要有瀏覽器即可使用：

### 步驟一：開啟系統
直接使用瀏覽器（推薦使用 Google Chrome 或 Microsoft Edge）雙擊開啟專案資料夾中的 `index.html`。

### 步驟二：匯入資料
將公文系統匯出的 Excel 檔案（`.xlsx` 或 `.xls`）**直接拖曳**至上方「資料匯入與篩選」區塊，或點擊「選擇檔案」進行上傳。
> **💡 提示：** 匯入後資料會自動存檔，下次打開同一個瀏覽器會自動載入上次的進度。

### 步驟三：設定條件與篩選
- 確認左上角的「**稽催基準日**」是否正確（預設為今天）。
- 利用「**公文類型**」或「**部門分頁**」快速切換您想查看的資料視角。
- 如果有找尋特定案件的需求，可直接在「**關鍵字搜尋**」輸入編號或主旨。
- 若系統分類的收發文不符合預期，可調整「**類型判斷基準**」天數。

### 步驟四：檢查與修改公文類型
查看清單中的「**最終類型**」欄位：
- 藍色標籤代表「收文」
- 黃色標籤代表「發文」
若有不符的案件，可以直接點擊該標籤，透過下拉選單手動強制切換。

### 步驟五：產生報表與輸出
確認畫面上的資料無誤後，可利用右上角的動作按鈕進行輸出：
- **[複製查詢結果]**：點擊後直接到 Outlook 或通訊軟體按下 `Ctrl+V` 貼上，表格格式會完整保留。
- **[匯出 Word]**：點擊下載 `.doc` 檔案，打開即可直接作為書面稽催報告。
- **[匯出 Excel]**：點擊下載 `.xlsx` 檔案，作為電子檔歸檔保存。

### 🧹 資料清除
如果想結束作業，或者要載入完全無關的全新檔案以避免舊資料干擾，請點選 **[清除資料]**，這會將畫面上與暫存在瀏覽器中的資料完全清空還原。
